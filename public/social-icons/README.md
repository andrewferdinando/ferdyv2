# Social Media Icons

Place your social media platform icon images in this directory.

## Required Files

- `facebook.png` - Facebook logo (40×40px recommended)
- `instagram.png` - Instagram logo (40×40px recommended)  
- `linkedin.png` - LinkedIn logo (40×40px recommended)
- `tiktok.png` - TikTok logo (40×40px recommended)
- `twitter.png` - Twitter/X logo (40×40px recommended)

## Image Specifications

- **Format**: PNG with transparent background
- **Size**: 40×40px recommended
- **Aspect Ratio**: Maintain original logo proportions (use `object-contain` CSS)

If images are not found, the system will automatically fall back to SVG icons.
